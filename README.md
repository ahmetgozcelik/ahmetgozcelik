### About Me

I am a Software Engineer dedicated to continuous learning and technical growth. While building robust backend and desktop applications with the **.NET Ecosystem**, I am deeply interested in **Embedded Systems** and **Defense Technologies**.

My current focus involves bridging high-level software architectures with low-level hardware communication.

**Key Focus Areas:**
* **Embedded Systems:** Worked with C++, Codesys, and communication protocols such as J1939, CAN Bus, Modbus, UART, SPI, and I2C.
* **Backend Development:** Building scalable APIs with ASP.NET Core.
* **Desktop Development:** Creating responsive UI/UX with WPF.
* **Architecture:** Applying Clean Architecture and MVVM principles.
